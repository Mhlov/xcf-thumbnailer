# XCF Thumbnailer

Thumbnailer for GIMP xcf files.

## INSTALLATION

### Dependecies

* gimp
* perl

```
sudo apt-get install gimp perl
```

### Installation

```
mkdir -p $HOME/.local/share/thumbnailers
cp bin/xcf-thumbnailer.pl $HOME/bin/
cp share/xcf.thumbnailer $HOME/.local/share/thumbnailers/
```
